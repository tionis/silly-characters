# SillyCharacters Client

React + Vite frontend for SillyCharacters.

## Responsibilities

- Nextcloud login UX
- card browsing/filtering/editing UI
- user settings and view preferences
- live updates via server SSE events

## Runtime

Development:

```bash
npm run dev
```

Build:

```bash
npm run build
```
